# Topologi Jaringan Cisco Packet Tracer

Berikut adalah topologi jaringan yang saya buat menggunakan Cisco Packet Tracer:

![Topologi Jaringan](b14a79c8-e65d-471d-b2ac-063ada6eb725.png)

## Deskripsi Singkat
Topologi ini terdiri dari:
- 1 Router utama terhubung ke modem
- 1 Switch pada ruangan 1
- 2 Access Point (Ruangan 2 dan Ruangan 3)
- Beberapa PC, Laptop, Tablet, dan Smartphone yang terhubung secara wireless

## Tujuan Project
- Melatih pemahaman konfigurasi Router, Switch, dan Access Point
- Implementasi jaringan campuran (wired & wireless)
- Pengujian komunikasi antar device di berbagai ruangan
